## How to use

1. 在根目录执行 `yarn` 安装依赖
2. 查看每章内容

    - 在 `src` 目录下查看每一章的内容，总结都在开头的注释
    - 在每一章下面执行 `npx tsc -t ES5` 后生成 `index.js` 文件，然后 `node index.js` 查看输出
